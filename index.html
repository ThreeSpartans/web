CMKeypad

This class is created for custom keypad view, default view is having numeric keys (0-9 & ".") and delete button.
 
It can be used in 2 ways.
  1. Default usage bu changing properties
  2. Create subclass and customize it
 
1  Default usage by changing properties
 
We can customize UI of keys with below attributes
Attribute Datatype Description
columns Integer Indicated number of keys in a single row
textColor color Text color of key
fontSize dimension(sp) Font size of key
shape custom
(oval/rectengle)		 Shape of key's background drawable
(rectangle is default)
keyPadding dimension(dp) Padding around key text
borderWidth_ dimension(dp) Width of border drawn around key
borderColor color Color of border drawn around key
radius dimension(dp) Corner radius of border
keyBackgroundColor color Background color for individual key
keyBackgroundColorPressed color Background color in pressed state
gradientStartColor color Start color of gradient background
gradientEndColor color End color of gradient background
gradientAngle custom
(top_to_bottom
bottom_to_top
left_to_right
right_to_left)
Angle of gradient
(left_to_right is default)
 
All above attributes are open so you can change it programatically also.
Call refresh() function after changing any variable to reflect changes
 
Using Custom values for keys
You can use below method to load custom values for keys. 
fun loadKeys(keys: ArrayList<KeysModel>)
 
Key Press callback
Use 
setKeyPressListener()
method for  key press callback, you will get KeysModel in callback method
 
2. Create subclass and customize it
 
 
Extend CMKeypad class and override below methods for further customization
open fun getView(columnPosition: Int, weight: Float): LinearLayout{​​}​​​​​​​​​
open fun bindView(view: View, columnPosition: Int, data: ArrayList<KeysModel>){​​​​​​​​​}​​​​​​​​​
 
Example:-
 class CustomKeypad : CMKeypad {​​​​​​​​​
    constructor(context: Context) : super(context)
    constructor(context: Context, attrs: AttributeSet) : super(context, attrs)
    constructor(context: Context, attrs: AttributeSet, defStyleAttr: Int) : super(context, attrs, defStyleAttr)

    override fun getView(columnPosition: Int, weight: Float): View {​​​​​​​​​
        val view = LayoutInflater.from(context).inflate(R.layout.layout_key, null)
        return view;
    }​​​​​​​​​

    override fun bindView(view: View, columnPosition: Int, data: ArrayList<KeysModel>) {​​​​​​​​​
        val mTextView: TextView = view.findViewById(R.id.tvKey)
        mTextView.setText(data.get(columnPosition).text)

    }​​​​​​​​​
}​​​​​​​​​

CMGradientDrawableBuilder

This class can be used to create solid as well as gradient color background drawables which can be used with with any view
 
  • Code snippet
var gdNormalState = GradientDrawable(
        GradientDrawable.Orientation.TOP_BOTTOM,
        intArrayOf(gradientStartColor, gradientEndColor))
gdNormalState?.shape = if (shape == 1) GradientDrawable.RECTANGLE else GradientDrawable.OVAL
gdNormalState?.cornerRadius = radius
gdNormalState?.setStroke(borderWidth.toInt(), borderColor)

val gDPressed = CMGradientDrawableBuilder()
        .setColor(backgroundColorPressed)
        .setShape(if (shape == 1) GradientDrawable.RECTANGLE else GradientDrawable.OVAL)
        .setCornerRadius(radius)
        .setStroke(borderWidth.toInt(), borderColor)
        .build()

val res = StateListDrawable()
res.addState(intArrayOf(android.R.attr.state_pressed), gDPressed)
res.addState(intArrayOf(), gdNormalState)
(this as AppCompatButton).setBackground(res)

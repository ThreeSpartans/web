CMKeyPad

This component is a customisable keypad, where we can have keys like (0 to 9, ".", "Delete (Back Space)") and we can customise the keys position. color of keys and background of it and etc.

CMButton

This is a customisable button, where we can change the font color, font type and set background programmatically and it is basically designable class, so we can do all the updations in storyboard itself .

 

Customisable Properties

 

  • Set the corner radius
  • Set the border width
  • Set the border color
  • Set the shadow color
  • Set the shadow offset
  • Set the shadow opacity
  • Set the shadow radius
  • Set the font color
  • Set the font type
  • Set the background color
  • Set the backgroung image
  • Set the Character spacing
  • Set the gradient background (start/end color,  start/end points, gradient angle)
  • Set html text

This is a customisable button class(IBDesignable). We can set corner radius, border, shadow, gradient background and a different shapes like oval and unique corners(different corner radius for all the four corners) programatically as well as using attribute inspector in storyboard/xib.

CMTextInput

This component is 4 in one, it can be used for Plain Text Input, Password, SSN & ZIP input. There are various attributes using which we can change behvaiour and UI of this Component. 

 

Below are few examples of CMTextIput

 

1) Plain Text (Default)

 

<com.conduent.apollo.ui.CMTextInput

    android:id=+id/cm_text_input"

    style=style/InputTextTheme"

    android:layout_width=atch_parent"

    android:layout_height=rap_content"

    android:imeOptions=ctionDone"

    android:inputType=extCapWords"

    android:fontFamily=asual"

    app:regex=a-zA-Z0-9._-]+@[a-z]+\\.+[a-z]+"

    app:regexMessage=lease enter valid email"

    app:isValidateOnFocusChange=rue"

    app:isMandatory=rue"

    app:min="

    app:max=0"

    app:minMessage=lease enter min 2 characters "

    app:maxMessage=ax is 10 characters"

    app:isMandatoryMessage=lease valid data."

    app:imeOptions=ctionDone"

    app:isEnabled=rue"

    app:isLabelAndErrorSameColor=alse"

    app:isMultiLine=alse"

    android:hint=pollo TextInput" />
